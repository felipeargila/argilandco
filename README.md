# Argila&Co Experience V5

## Ajustes desta versão

- Texto: "Toque e encaixe os círculos."
- Grid muito menos aparente.
- Círculos-alvo em pontilhado bem discreto.
- Ao encaixar, o texto aparece pequeno dentro do círculo.
- Círculos entram caindo do topo e quicam na base.
- Círculos soltos se movem com o sensor do celular.
- Geometria mantida:
  - viewBox 500x500
  - raio 150
  - centros em 150/150, 350/150, 350/350, 150/350
  - símbolo formado pelas interseções reais.
